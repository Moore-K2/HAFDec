# HAFDec
[PRCV] Self-supervised monocular depth estimation with HAFDecoder and HFSA attention. Lightweight (2.5M params, 5.5 GFLOPs) achieving Abs Rel 0.107 on KITTI and strong zero-shot generalization on Make3D.
